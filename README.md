# Smart Classroom

A modern classroom management platform connecting teachers and students.

## Features

- **Authentication**: Separate signup/signin for teachers and students
- **Teacher Features**:
  - Create and manage classes
  - Add daily class minutes with announcements and tasks
  - Approve/reject student join requests
  - Manage enrolled students
  
- **Student Features**:
  - Browse and request to join classes
  - View class minutes by date
  - Access historical class content
  - See announcements and tasks

## Installation
```bash
npm install
```

## Run Development Server
```bash
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000)

## Tech Stack

- React 18
- Lucide Icons
- Tailwind CSS (utility classes)
- Claude's color theme
